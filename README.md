# My Portfolio

Access my portfolio website at https://katelynkunzmann.github.io/my-portfolio/

## How to build and deploy with vite

`npm run dev` -> "vite"  
`npm run build` -> "vite build"  
`npm run preview` -> "vite preview

### For GitHubPages

`npm run deploy`
